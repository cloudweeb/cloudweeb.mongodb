Cloudweeb Mongodb
=========

Ansible role to install MongoDB

Requirements
------------

None

Role Variables
--------------

```YAML
# MongoDB version that want to be installed
mongodb_version: 4.0
```

Dependencies
------------

None

Example Playbook
----------------

    - hosts: servers
      roles:
         - role: cloudweeb.mongodb
